# Phase 12 -> Phase 13 交接

## 当前完成进度

- 自定义确认弹窗新增轻量 focus trap。
- `handleConfirmKeydown()` 现在同时处理 Esc 和 Tab：
  - Esc 关闭确认弹窗。
  - Tab 从最后一个按钮循环到第一个按钮。
  - Shift+Tab 从第一个按钮循环到最后一个按钮。
  - 焦点在弹窗外时，Tab 会拉回弹窗内。
- `tests/frontend_smoke.js` 扩展焦点模拟：
  - `focus()` 更新 `document.activeElement`。
  - `pressDocumentKey()` 支持 `shiftKey`。
  - 新增焦点循环 smoke。
- README 和开发方案同步更新了自定义确认弹窗和焦点循环覆盖范围。
- 已梳理其他浮层：
  - 媒体查看器缺少 Esc 关闭、焦点恢复和焦点循环。
  - composer 面板是内联展开区域，不是遮罩式浮层，优先级较低。
