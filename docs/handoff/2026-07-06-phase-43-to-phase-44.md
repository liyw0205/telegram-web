- 媒体查看器准备已增加前端忙碌保护：
  - `openMediaViewerByMessage()` 按消息 ID 复用准备中的 Promise。
  - 同一媒体准备中重复点击显示“媒体正在准备，请稍候”。

  - 重复触发显示“媒体下载任务正在创建，请稍候”。
  - 任务创建成功或失败后恢复按钮状态；失败时查看器保持打开并可再次重试。
- 前端 smoke 已覆盖媒体准备重复点击、准备失败重试、下载任务创建重复触发、失败恢复按钮状态。
