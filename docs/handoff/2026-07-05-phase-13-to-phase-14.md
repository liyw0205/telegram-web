# Phase 13 -> Phase 14 交接

## 当前完成进度

- 媒体查看器新增键盘交互和焦点恢复。
- `openGallery()` 现在会：
  - 记录打开前焦点。
  - 显示查看器。
  - 绑定 document keydown。
  - 渲染当前媒体。
  - 聚焦关闭按钮。
- `closeGallery()` 现在会：
  - 隐藏查看器。
  - 解绑 document keydown。
  - 恢复打开前焦点。

  - Esc 关闭。
  - ArrowLeft 上一项。
  - ArrowRight 下一项。

- `tests/frontend_smoke.js` 增加媒体查看器 mock DOM 和键盘 smoke。
- README 和开发方案同步更新覆盖范围。
