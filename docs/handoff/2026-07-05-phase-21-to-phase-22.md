# Phase 21 -> Phase 22 交接

- `/diagnostics` 可访问性和状态语义已补强：
  - 页面 section 关联主标题。
  - 刷新按钮增加 `aria-label` 和 `title`。
  - 摘要区域使用 `role="status"`、`aria-live="polite"`、`aria-atomic="true"` 和动态 `aria-busy`。
